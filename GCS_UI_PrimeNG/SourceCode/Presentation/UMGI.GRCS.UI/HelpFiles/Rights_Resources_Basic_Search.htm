Rights Resources Basic Search

Only users with View Repertoire Rights/View Repertoire Rights Sensitive data’  privileges can search and view the rights data created for a release/resource/track in GRS. Using the Rights Resources Basic Search, user can review the rights for required resources. You can click the following accordions to enter the respective fields for reviewing the rights of the required resources.

  • Resource Parameters

  • Resource Rights Parameters

  1. Select Repertoire Rights Search -> Rights -> Show Resources -> Basic Search. The Rights Resources Basic Search screens displayed. By default, the ‘Resource Parameters’ section is displayed. You can enter one or a combination of criteria to search for the resource (s).

 

 

Resource Parameters

  1. ISRC – You can manually enter an ISRC in the free text field which is an Exact search in GRS

  2. Also, you can browse and upload an ISRC by clicking the  icon .You can browse a spread sheet locally having multiple ISRCs in first column and upload it to search for the rights data of the resources in GRS having those ISRCs

Note: All other search parameters would be disabled if user provides an ISRC manually or uploads a list of ISRCs via excel upload

  1. Resource Title – Enter a Resource title  in the free text box; the system then automatically searches and display a list of Resource Title that match the text entered.

  2. Resource Type – Select 'Audio/Video/Image' from the resource type drop-down as required. By default system displays an 'Empty' value.

  3. Artist Name – A Free text field to enter artist name on the resource and system automatically searches and display a list of Artists which contains  the entered text.

  4. Exact Search – Select this check box to set the Artist Name field as exact match search field.

  5. Linked Contract Name – A Multi line free text box to search a linked contract. Click the search icon to open Search for Contract pop-up.

  6. Following search parameters are available in the pop-up.

    • Artist - A free text box with contains search in GRS. Along with the text box, an exact check box is also displayed; if user checks the 'exact' check box, system will do an exact search in GRS

    • Contract ID - A free text box with exact search in GRS

    • Local Contract Reference Number- A free text box with exact search in GRS

    • Contracting Party - A free text box with contains search in GRS

    • Clearance Admin Company - An Auto fill search in GRS, you have to provide atleast 3 characters for auto fill option to get executed.

  7. Select or enter relevant information and search for a linked contract.

  8. Select a linked contract, the pop-up is closed and selected contract is displayed in the field. The linked contract name includes Artist name, clearance admin company, and commencement date separated by comma.

  9. Repertoire linked to multiple contracts – If you provide one or more linked contracts, then this check box is disabled.

  10. Click the ‘Resource Right’s Parameters’ header to display the search fields in this section

Resource Rights Parameters

 

  1. Territorial Rights – Click the Select/Remove Countries hyperlink to open the Select/Remove Countries pop-up to select one or more countries for search.

  2. The pop-up displays list of countries excluding territories. You can select one or more countries by clicking the check box adjacent to countries you wish to select.

  3. The selected countries are displayed in the ‘Selected Countries’ section adjacent to the countries list.

  4. You can remove the selected countries by clicking the corresponding [x] icon in the Remove All section.

  5. After selecting one or more countries and clicking on OK, pop-up gets closed and the selected countries are shown as comma separated in the Territorial Rights text box.

Note: If user selects multiple countries and clicks on search then those applicable repertoire items should be displayed which have rights to be exploited in ALL countries selected by user. For example: If user selects France & Italy, it should return rows where they have rights to be exploited in both France AND Italy.

  1. Rights Review Status - Select a 'New-For-Review/Final/Final-For-Review' rights review status from the drop-down.

  2. Rights Period - Set the contractual Rights Period using the available drop-down values. The Rights Period is set to 'Empty' by default.

  3. Lost Rights Indicator - Select ‘Yes or ‘No’ from the drop-down.

  4. Physical Rights Exploitation - Select ‘Yes or ‘No’ from the drop-down.

  5. Lost Rights Reason – Set the Lost Rights Reason using the available drop-down values.

Note:You can select a lost rights reason and view it in the Search Results section irrespective of your privileges, that is, all the users of GRS can select and view the lost rights reason.

  1. Digital Exploitation Rights - Select ‘Yes or ‘No’ from the drop-down.

  2. Digital Unbundling Allowed - Select ‘Yes or ‘No’ from the drop-down.

Note: If you have selected ‘No’ in the Digital Exploitation Rights dropdown, then this filed is disabled.

  1.  Mobile Exploitation Required - Select ‘Yes or ‘No’ from the drop-down.

  2.  PPB Revenue Claim - Select ‘Yes or ‘No’ from the drop-down.

  3.  Active for Marketing - Select ‘Yes or ‘No’ from the drop-down.

  4.  Lost Rights Date - Select from date and to date from the respective calendar lookups.

  5. Click Search- The Resource Rights repertoire results that match the search criteria specified are displayed as list in the Search Results grid. The search Results grid is displayed as Paged format.

  6. Click Reset- All the search criteria entered will be cleared, ready for entering new search criteria.

Search Result

  1. The list of resources matching the search criteria is displayed as a table on screen. Each row displays the following Resource Rights details:

    • Traits

    • ISRC

    • UPC

    • Artist

    • Title

    • Version Title

    • Release Date- Earliest release date across all releases which resource is associated to

    • Linked Contract

      1. Shown as a linked contract icon in grid, with a tool tip as- Concatenation of Artist or Contracting Party if no Artist, and Contract Effective Date.

      2. When the linked contract is a Split contract, then icon for split contract is displayed with a tool tip same as above.

    • Clearance Admin Company

    • P Year

    • Review Status

    • Active for Marketing

    • Territorial Rights

      1. If the length of countries exceeds those maximum characters, then you can point the cursor on the country to view the complete TRs in a tool tip.

    • Rights Period

    • Lost Rights

    • Lost Rights Reason

    • Lost Rights Date

    • Physical Exploitation Rights

    • Digital Exploitation Rights

    • Mobile Exploitation Rights

    • PPB Revenue Claim

  2. You select one or more rows from the search results and clicks on Review Rights Button then system ports the selected item (s) to the Custom Resource Rights Review Work Queue.

  3. Search results can be exported to an Excel document by clicking the  button.
