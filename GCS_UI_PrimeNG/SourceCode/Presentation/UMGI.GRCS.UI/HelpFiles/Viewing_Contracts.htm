Viewing Contracts

*Note: Contract viewing and editing rights are only assigned to specific users, and limited by your security access.

After searching for a contract, you can view its details in the View Contract screen.

  1. From the search result grid, select the check box corresponding to the required contract.

  2. Click View Contract. The View Contract screen is displayed with the details of the selected contract.

Note: If you do not have the required security settings to view a contract, an error message will be displayed.
