Contract Maintenance Work Queue

Any contracts which require your attention before completion are displayed in the Contract Maintenance Work Queue. Contracts displayed in this work queue are filtered according to your Role and Security Settings.

Viewing Contracts in the Work Queue

  1. Select Contract Administration -> Contract Maintenance Work Queue. The Contract Maintenance Work Queue screen is displayed.


     

  1. The list of contracts is displayed automatically in the Contract Maintenance grid when this screen is accessed.

  2. Each row in the Contract Maintenance grid displays details such as ‘ Workflow Status’, ‘Contract Status’, ‘Artist Name’, ‘Contract Description’, ‘Admin Company’ and an indication () whether Repertoire is already linked . The list of contracts in Contract Maintenance grid can be sorted in ascending or descending alphanumeric order by clicking each column header.
