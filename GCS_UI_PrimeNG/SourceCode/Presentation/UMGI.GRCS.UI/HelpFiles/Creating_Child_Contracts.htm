Creating Child Contracts

You can create Parent and Child relationships between contracts. This enables contracts to be linked, and any future changes to a Parent Contract can then automatically be propagated down to any related child contracts.

To create a Child Contract:

  1. Search for a contract, and open it to the Edit Contract screen.

  2. Click Create Child Contract and a message is displayed prompting you to propagate the rights from the selected ‘Parent’ contract to the new child contract.

  3. If you click Yes in the Message box, GRS creates and displays the new child contract with the original contract displayed in the parent contract section. The Parent Contract rights data will be propagated to the new Child Contract; however you can edit the child’s rights data as necessary.

  4. If you click No in the Propagation Message box, GRS still creates and displays the new child contract with the original contract displayed as the parent contract. However the Parent Contract rights data are not propagated to the new Child Contract. You can edit the child’s rights data as necessary.


     

  1. For more information for creating and completing new contracts, refer ‘Create a New Contract’.

Notes:

  1. The Create Child Contract button is disabled if the selected contract is already a child contract.

  2. The Create Child Contract button is displayed only for users with the sufficient editing rights.
