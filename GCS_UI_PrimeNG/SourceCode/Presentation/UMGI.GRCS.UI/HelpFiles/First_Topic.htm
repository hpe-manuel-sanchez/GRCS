Welcome

Welcome to the Contract Administration Help System, a ''Screen Context Sensitive Web Help'' of UMG Global Rights System (GRS). The conceptual and procedural information for executing various tasks using 'Contract Administration' module are detailed in the help system.

You can view help files related to each screen by clicking and selecting the Help option from the drop-down.


The help file specific to a screen is integrated with the screen. For example, to view information for creating a contract, you need to access the Create New Contract screen and select the Help option.
