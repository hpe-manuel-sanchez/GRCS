Change the Linked Contract from the Release

Only users with Edit Rights data (Header), Edit Rights data (Digital),Edit Rights data (Secondary), Edit Rights data (Pre-Cleared), Edit Rights Review Status privileges can search, view and Amend the rights data created for a resource/track in GRS. Using the Custom Rights Review WQ  user can review/amend the rights for required releases.

  1. Select Rights Administration -> Rights Review Work Queue-> Create Custom Settings -> Releases. The Custom Setting page is displayed. By default, the ‘Custom Parameter’ Tab is displayed. You can enter one or a combination of criteria to create the Custom Work queue for the releases.

  1. On clicking the Change Link button after selection of the linked Resource(s) from the Work queue Grid, System presents a pop up screen for user to select an existing contract in GRS to replace the previous contract linked with the selected releases

  1. User select a single resource from WQ to change link then by default on the pop-up, system displays a list of contracts created for same artist as existing on the resource by those Clearance Admin Company (s) which user has access to in his security profile for linking privilege.

  1. User select release (s) from WQ to change link and selects any approved contract to change the linked contract.

  2. Additionally, user can also search any other contract using various parameters like:

    • Contract ID

    • Local Contract No

    • Artist

    • Contracting Party

    • Clearance Admin Company

  1. ‘Change Link’ action is not applicable for resources in WQ having ‘reason for review’ as ‘Contract Linking Required’.
