Mass Update Release Rights Data Sets by Bulk edit

Only users with Edit Rights data (Header), Edit Rights data (Digital),Edit Rights data (Secondary), Edit Rights data (Pre-Cleared), Edit Rights Review Status privileges can search, view and Amend the rights data created for a release (s) in GRS. Using the Custom Rights Review WQ  user can review/amend the rights for required releases.

  1. Select Rights Administration -> Rights Review Work Queue-> Create Custom Settings -> Releases. The Custom Setting page is displayed. By default, the ‘Custom Parameter’ Tab is displayed. You can enter one or a combination of criteria to create the Custom Work queue for the releases.

 

  1. You can amend the Rights Data sets by Bulk Edit options for below tabs:

    • Rights Acquired

    • Territorial Rights

    • Digital Restrictions

Bulk Edit Rights Acquired Tab

  1. On Clicking “Bulk- edit” Button on the Custom Release Rights Review Screen triggers the popup for Bulk-editing provided rows are selected for mass updates.

 

  1.  Bulk Edit- Rights Acquired popup contains the editable fields of the Rights Acquired Tab enlisted below:

[Each of these parameters have drop down values as in the grid and additionally a default value as “No Change Required” either as text or as icons only for Rights Exception Applied]

    • Active for marketing

    • Rights Period

    • Lost Rights?

    • Lost Rights Reason

    • Lost Rights date

    • Rights Exception Applied

    • Lost Rights Reason

    • Physical Exploitation

    • Digital Exploitation

    • Digital Unbundling Allowed

    • Mobile Exploitation

    • PPB Revenue Claim

  1. You select values in the grid as necessary for Bulk Updating and clicks on Apply Changes which is used for applying the changed values to the selected rows

  2. You select values in the grid as necessary for Bulk Updating and clicks on Cancel to exit from the popup and navigates back to the Grid

  3. You select values in the grid as necessary for Bulk Updating and clicks on Reset Selection which refreshes the field values and restores it to the default value “No Change Required”

Bulk Edit – Territorial Rights

  1. On clicking on the bulk dit button on the Rights Acquired Tab, if you want to bulk edit the Territorial Rights they need to navigate to the adjacent tab in the pop up

  1. The system displays Review Status atop the pop with default value of “No Change Required”. You can edit the Review Status from this tab as well.

  2. You select values in the grid as necessary for Bulk Updating and clicks on Apply Changes which is used for applying the changed values to the selected rows

  3. You select values in the grid as necessary for Bulk Updating and clicks on Cancel to exit from the popup and navigates back to the Grid

  4. You select values in the grid as necessary for Bulk Updating and clicks on Reset Selection which refreshes the field values and restores it to the default value “NA”

Bulk Edit- Digital Restrictions

  1. On clicking on the bulk edit button on the Digital Restrictions  Tab on the main grid then the system navigates the user to the Bulk Edit Digital Restriction Popup

  1. By default, system displays the Change Restriction tab, Add Restriction Tab, Remove Restriction Tab and displays the screen with below attributes

    • Use Type

    • Commercial Model

    • Restrictions

    • Restriction Reasons

    • Consent Period

    • Notes

  1. You select values in the grid as necessary for Bulk Updating and clicks on Apply Changes which is used for applying the changed values to the selected rows

  2. You select values in the grid as necessary for Bulk Updating and clicks on Cancel to exit from the popup and navigates back to the Grid

  3. You select values in the grid as necessary for Bulk Updating and clicks on Reset Selection which refreshes the field values and restores it to the default value “No Change Required”
