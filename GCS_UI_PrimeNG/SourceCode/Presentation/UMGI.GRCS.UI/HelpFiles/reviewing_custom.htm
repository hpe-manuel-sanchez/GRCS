Reviewing the Rights Data sets for Resources/Tracks

Only users with Edit Rights data (Header), Edit Rights data (Digital),Edit Rights data (Secondary), Edit Rights data (Pre-Cleared), Edit Rights Review Status privileges can search, view and Amend the rights data created for a resource/track in GRS. Using the Custom Rights Review WQ , user can review the rights/amend for required resources/tracks.

  1. Select Rights Administration -> Rights Review Work Queue-> Create Custom Settings -> Resources & Tracks. The Custom Setting page is displayed. By default, the ‘Custom Parameter’ Tab is displayed. You can enter one or a combination of criteria to create the Custom Work queue for the resource (s)/ track (s).

  1. The Following Tabs can be accessed to Review/Amend the Rights for required Resource (s) / Track (s) after creating the custom WQ

    • Rights Acquired Tab

    • Digital Restriction Tab

    • Secondary Exploitation Tab

    • Pre- Clearance Tab

 

Rights Acquired Tab

 

  1. The list of resources matching the search criteria is displayed as a table on screen. Each row displays the following Resource Rights details:

    • Traits

    • ISRC

Note: System displays the ISRC as a link, on clicking on the ISRC ID user navigates to the Custom Release Rights Review Work Queue containing all the releases the ISRC is part of.

    • UPC

Note: System displays the UPCs only for track data items. It is identified as a multi-artist release or otherwise as a normal release. On clicking the UPC the user is navigated to the custom release rights review WQ for that UPC

    • Artist

    • Title

    • Version Title

    • Release Date- Earliest release date across all releases which resource is associated to

    • Linked Contract

      • Shown as a linked contract icon in grid, with a tool tip as- Concatenation of Artist or Contracting Party if no Artist, and Contract Effective Date.

      • When the linked contract is a Split contract, then icon for split contract is displayed with a tool tip same as above.

    • Clearance Admin Company

    • Side Artists Exists

    • Sample Exists

    • P Year

    • Rights Type

    • Reason For Review

    • Territorial Rights

      • If the length of countries exceeds those maximum characters, then you can point the cursor on the country to view the complete TRs in a tool tip.

    • Error

    • Active for Marketing

    • Rights Period

    • Lost Rights

    • Lost Rights Reason

    • Lost Rights Date

    • Rights Exception

    • Physical Exploitation Rights

    • Digital Exploitation Rights

    • Mobile Exploitation Rights

    • PPB Revenue Claim
