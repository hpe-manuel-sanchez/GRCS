Creating Address Books

Email Notification Recipient groups are managed within Address Books. Address books are assigned to contracts, so specific users can be automatically kept informed of any impending contractual key dates or tasks Creating A New Address Book

  1. Select Contract Administration -> Manage Address Book. The Manage Address Book screen is displayed. The Filter Role/Group Name grid displays the list of existing recipient groups. The grid displays the Name, Country and Email IDs of each group.


     

  1. Click Create New Entry. The Add Role/Group pop-up is displayed.


     

  1. Role/Group Name - Enter a unique name in the Role/Group Name text box.

  2. Country - Specify the country or countries by searching and selecting in the Auto Suggest look-up box.

  3. Email IDs - Enter all Email addresses of the group members, separated by a comma or semi-colon.

  4. Click Save. The Add Role/Group pop-up is closed and the newly added group is listed in the grid.
