Unlink the contract from Linked Resource

  1. Select Rights Administration -> Rights Review Work Queue. The Rights Review Filter accordion is displayed. By default, the ‘Rights Review Filter’ section is displayed. You can enter one or a combination of criteria to search for the resource (s)/ track (s).

  1. Select the resource from the Work queue and click on 'Unlink Contract Link’ button at the bottom of the work queue grid

  1. The System prompts user to ask his confirmation Unlinking the selected resource from their contract will remove their rights data from the system. Do you want to continue?”

  2. When User chooses ‘Yes’, the System deletes that contract-resource linking & the rights data of the selected resource (s) derived from that contract.

  3. When User chooses ‘No’, then no action is taken and user navigates back to the Priority work queue screen.
