Creating Contracts Using Templates

  1. Select Contract Administration -> Create New Contract from Template. The Choose Template pop-up is displayed.


     

  1. Select a template from the drop-down.

  2. Click OK. The Choose Template pop-up is closed and the Create New Contract from Template screen is displayed with details of the selected template.

  3. You can then edit the required contract information to create a new contract. To create a new contract without using a template, refer to the ‘Create A New Contract’ section.
