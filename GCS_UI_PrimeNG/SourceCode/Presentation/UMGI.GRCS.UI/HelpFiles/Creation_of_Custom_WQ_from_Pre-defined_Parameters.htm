Creation of Custom WQ from Pre-defined Parameters

• Only users with Edit Rights data (Header), Edit Rights data (Digital),Edit Rights data (Secondary), Edit Rights data (Pre-Cleared), Edit Rights Review Status privileges can search, view and Amend the rights data created for a resource/track in GRS. Using the Custom Rights Review WQ  user can review the rights/amend for required resources/tracks. You can click the following accordions to enter the respective fields for reviewing the rights of the required resources.

  1. Select Rights Administration -> Rights Review Work Queue-> Create Custom Settings -> Resources & Tracks. The Custom Setting page is displayed. By default, the ‘Custom Parameter’ Tab is displayed. You can enter one or a combination of criteria to create the Custom Work queue for the resource (s)/ track (s).

 

  1. System displays the predefined parameters as second tab and User selects the predefined parameter tab to create the custom Work queue by clicking on Create Work queue button.

 

  1. You select the predefined parameters from the below list:

    • Resource Rights Data sets where the earliest release date is less than today’s  date +n weeks

    • Track Rights Data sets where the earliest release date is less than today’s  date +n weeks

    • Track & Resource Rights Data sets where the earliest release date is less than today’s  date +n weeks

Note: Below parameters requires Release Date parameter range with the respective predefined parameter

    • Track Rights Data sets for tracks on releases which are not Multi-Artist Compilations

    • Rights Data Sets for Resources with Samples

    • Rights Data Sets for Resources with Side Artists

    • Rights Data Sets for Resources with Digital Restrictions marked as Consent Required, Consult or Refer to Legal

  1. You select either Release Date Range by selecting from and to dates by clicking on the calendar control or can only include Repertoire with no release dates

  2. By clicking on Reset Selection, system refreshes the field values and restores it to the default value as empty blank cells.
