Copying Contracts

After searching for a contract, you can copy it to create a new contract.

  1. Search for the required contract in the Search Contract screen.

  2. In the search result grid, select the check box corresponding to the contract row to copy.

  3. Click Copy Contract. A new contract is created and displayed in the Create New Contract screen. The contract details will be copied from the originally selected contract and the contract workflow status will be reset to ‘Data Entry’.


     

  1. You can complete or edit any required fields to complete the new contract, and then press Save. The new contract will be created and given a new unique Contract ID number. For more information on contract creation, refer to ‘Create New Contracts’.

Note: The Copy Contract button is only available to users with the sufficient editing rights.
