Creation of Custom Release WQ from Custom Parameters

Only users with Edit Rights data (Header), Edit Rights data (Digital),Edit Rights data (Secondary), Edit Rights data (Pre-Cleared), Edit Rights Review Status privileges can search, view and Amend the rights data created for a release (s) in GRS. Using the Custom Release Rights Review WQ  user can review/amend the rights for required releases.

  1. Select Rights Administration -> Rights Review Work Queue-> Create Custom Settings -> Releases. The Custom Setting page is displayed. By default, the ‘Custom Parameter’ Tab is displayed. You can enter one or a combination of criteria to create the Custom Work queue for the releases.

 

  1. You click on Create Custom Setting Link named ‘Resource & Tracks’ on the Priority Rights Review Work queue.

  2. By default, System presents the Custom Parameters Tab to create the Custom Work queue

  1. You select either Release Date Range by selecting from and to dates by clicking on the calendar control or can only include Repertoire with no release dates

  2. You enter one or more combination of following Parameters to create the Work Queue by clicking on Create Work queue button

  • UPC

  • Artist Name

  • Release Title

  • Clearance/Data Admin Company

  • Linked Contract

  • Release Date Range

  1. By clicking on Reset Selection, system refreshes the field values and restores it to the default value as empty blank cells.
