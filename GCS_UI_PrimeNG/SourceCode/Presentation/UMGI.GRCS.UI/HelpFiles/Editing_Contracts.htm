Editing Contracts

After searching for a contract, you can edit its details.

*Note: Contract editing rights are only assigned to specific users, and limited by security access. Contact your local security administrator for further information about your access rights.

  1. Search for the required contract in the Search Contract screen.

  2. In the search result grid, select the check box corresponding to the contract row to edit.

  3. Click Edit Contract. The Edit Contract widow is displayed with the details of the selected contract.

  4. Edit any required fields and click Save.


     

  1. For more information on editing contracts, refer ‘Editing Contracts in Work Queue’.
