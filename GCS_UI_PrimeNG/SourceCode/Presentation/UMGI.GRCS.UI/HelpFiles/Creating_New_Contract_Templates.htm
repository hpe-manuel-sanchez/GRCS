Creating New Contract Templates

When creating a new contract, you can save it as a new Contract Template, to be used as the basis of future contracts.

  1. Create a new contract from the Create New Contract screen.

  2. Complete the contract fields and settings as you require, then click Save as a Template. The Enter Contract Template Name pop-up is displayed.

Note: The minimum data required to create a Contract Template is a Data Admin Company-Country.


       

  1. Enter a unique template name in the text box.

  2. Click Save. The new contract template is saved.

Note: If the template name already exists within the same Clearance Admin Company then the system prompts you to enter different name.
