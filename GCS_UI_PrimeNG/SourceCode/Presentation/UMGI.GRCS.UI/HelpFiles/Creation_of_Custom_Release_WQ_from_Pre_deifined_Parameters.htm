Creation of Custom Release WQ from Pre defined Parameters

Only users with Edit Rights data (Header), Edit Rights data (Digital),Edit Rights data (Secondary), Edit Rights data (Pre-Cleared), Edit Rights Review Status privileges can search, view and Amend the rights data created for a release (s). Using the Custom Release Rights Review WQ  user can review/amend the rights for required releases.

  1. Select Rights Administration -> Rights Review Work Queue-> Create Custom Settings -> Releases. The Custom Setting page is displayed. By default, the ‘Custom Parameter’ Tab is displayed. You can enter one or a combination of criteria to create the Custom Work queue for the releases.

  1. System displays the predefined parameters as second tab and User selects the predefined parameter tab to create the custom Work queue by clicking on Create Work queue button.

  1. You select the predefined parameters from the below list:

    • Multi-Artist Compilation Release

    • OST Releases only

    • Non Mac Releases with Non- Exclusive tracks

  1. You select either Release Date Range by selecting from and to dates by clicking on the calendar control or can only include Repertoire with no release dates

  2. By clicking on Reset Selection, system refreshes the field values and restores it to the default value as empty blank cells.
