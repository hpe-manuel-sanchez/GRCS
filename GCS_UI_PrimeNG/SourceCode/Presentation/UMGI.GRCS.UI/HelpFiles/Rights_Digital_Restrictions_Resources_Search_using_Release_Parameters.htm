Rights & Digital Restrictions Resources Search using Release Parameters

Only users with View Repertoire Rights/View Repertoire Rights Sensitive data’  privileges can search and view the rights data created for a release/resource/track in GRS. Using the Rights Resources Search using Release Parameters, user can review the rights for required resources. You can click the following accordions to enter the respective fields for reviewing the rights of the required resources.

  • Release Parameters

  • Resource's Rights Parameters

  • Digital Restrictions

  1. Select Repertoire Rights Search -> Rights & Digital Restrictions -> Show Resources -> Search using Release Parameters. The Rights & Digital Restrictions Resources Search using Release Parameters screen is displayed.

  1. By default, the ‘Release Parameters’ section is displayed. You can enter one or more combination in the search criteria

Release Parameters

  1. UPC – Free text field with ‘Browse’ icon. You can enter an UPC.

  2. Also, you can browse and upload UPCs by clicking the    icon.

Note:All other search parameters would be disabled if user provides an UPC manually or uploads a list of UPCs via excel upload

  1. Release Title – Free text field to enter the release title.

  2. Artist Name – Free text field to enter artist name and search.

  3. Exact Search – Select this check box to set the Artist Name field as exact match search field.

  4. Linked Contract Name – Multi line free text box to search a linked contract. Click the search icon to open Search for Contract pop-up.

  5. Following search parameters are available in the pop-up.

    • Artist

    • Contract ID

    • Local Contract Reference Number

    • Contracting Party

    • Clearance Admin Company

  1. Select or enter relevant information and search for a linked contract.

  2. Select a linked contract, the pop-up is closed and selected contract is displayed in the field. The linked contract name includes Artist name, Clearance admin company, and commencement date separated by comma.

  3. Also, you can specify Resources Rights parameters as search criteria by entering or selecting information for the fields in the Resource’s Rights Parameters section.

Resource's Rights Parameters

 13. Territorial Rights – Click the Select/Remove Countries hyper link to open the Select/Remove Countries pop-up to            select one or more countries for search.

  1. The pop-up displays list of countries excluding territories. You can select one or more countries by clicking the check box adjacent to countries you wish to select.

  2. The selected countries are displayed in the ‘Selected Countries’ section adjacent to the countries list.

  3. You can remove the selected countries by clicking the corresponding [x] icon in the Remove All section.

  4. After selecting one or more countries and clicking on OK, pop-up gets closed and the selected countries are shown as comma separated in the Territorial Rights text box.

Note:If user selects multiple countries and clicks on search then those applicable repertoire items should be displayed which have rights to be exploited in ALL countries selected by user. For example: If user selects France & Italy, it should return rows where they have rights to be exploited in both France AND Italy.

  1. Rights Review Status - Select a status from the drop-down.

  2. Rights Period - Select a period from the drop-down.

  3. Lost Rights Indicator - Select ‘Yes or ‘No’ from the drop-down.

  4. Physical Rights Exploitation - Select ‘Yes or ‘No’ from the drop-down.

  5. Lost Rights Reason – Select a reason from the drop-down.

Note:You can select a lost rights reason and view it in the Search Results section irrespective of your privileges, that is, all the users of GRS can select and view the lost rights reason.

  1. Digital Exploitation Rights - Select ‘Yes or ‘No’ from the drop-down.

  2. Digital Unbundling Allowed - Select ‘Yes or ‘No’ from the drop-down.

Note:If you have selected ‘No’ in the Digital Exploitation Rights dropdown, then this filed is disabled.

  1. Mobile Exploitation Required - Select ‘Yes or ‘No’ from the drop-down.

  2. PPB Revenue Claim - Select ‘Yes or ‘No’ from the drop-down.

  3. Active for Marketing - Select ‘Yes or ‘No’ from the drop-down.

  4. Lost Rights Date - Select from date and to date from the respective calendar lookups.

  5. You can enter or select relevant information as search criteria in the Digital Restrictions section.

Digital Restriction

  1. Use Type – Select a use type from the drop-down

  2. Commercial – Select a option from the drop down

  3. Restriction Exists – Select this check box if digital restrictions are available.

  4. Restrictions Doesn’t Exists - Select this check box if digital restrictions are not available.

  1. To add an individual Digital Restriction row, click the Add Restrictions hyperlink. The Digital Restriction row is displayed.

  2. To remove a Digital Restriction row, click the Remove icon corresponding to the row you wish to remove.

  3. Click Search. The Resources Rights & Digital Restrictions repertoire results that match the search criteria specified are displayed as list in the Search Results grid. The search Results grid is displayed as Paged format.

  4. Click Reset- All the search criteria entered will be cleared, ready for entering new search criteria.

  1. The list of resources matching the search criteria is displayed as a table on screen. Each row displays the following Resource Rights details:

    • Traits

    • ISRC

    • UPC

    • Artist

    • Title

    • Version Title

    • Release Date- Earliest release date across all releases which resource is associated to

    • Linked Contract

      1. Shown as a linked contract icon in grid, with a tool tip as- Concatenation of Artist or Contracting Party if no Artist, and Contract Effective Date.

      2. When the linked contract is a Split contract, then icon for split contract is displayed with a tool tip same as above.

    • Clearance Admin Company

    • P Year

    • Review Status

    • Active for Marketing

    • Territorial Rights

      • If the length of countries exceeds those maximum characters, then you can point the cursor on the country to view the complete TRs in a tool tip.

    • Rights Period

    • Lost Rights

    • Lost Rights Reason

    • Lost Rights Date

    • Physical Exploitation Rights

    • Digital Exploitation Rights

    • Mobile Exploitation Rights

    • PPB Revenue Claim

    • Digital Unbundling Allowed

    • Use Type

    • Commercial

    • Consent Period

    • Notes

  3. You select one or more rows from the search results and clicks on Review Rights Button then system ports the selected item (s) to the Custom Resource Rights Review Work Queue.

  4. Search results can be exported to an Excel document by clicking the  button.
