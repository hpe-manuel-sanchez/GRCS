Rights Tracks Search Using Release Parameters

Only users with View Repertoire Rights/View Repertoire Rights Sensitive data’  privileges can search and view the rights data created for a release/resource/track in GRS. Using the Rights Tracks Search using Release Parameters, user can review the rights for required tracks. You can click the following accordions to enter the respective fields for reviewing the rights of the required resources.

  • Release Parameters

  • Track Rights Parameters

  1. Select Repertoire Rights Search -> Rights -> Show Tracks -> Search using Release Parameters. The Rights Tracks Search using Release Parameters screen is displayed. By default, the ‘Resource Parameters’ section is displayed. You can enter one or a combination of criteria to search for the track (s).

Release Parameters

  1. UPC – Free text field with ‘Browse’ icon. You can enter an UPC. Also, you can browse and upload UPCs by clicking the  icon

Note:If you provide an ISRC or upload a list of UPCs, all other search fields in this section are disabled.

  1. Release Title – Free text field to enter the release title.

  2. Artist Name – Free text field to enter artist name and search.

  3. Exact Search – Select this check box to set the Artist Name field as exact match search field.

  4. Linked Contract Name – Multi line free text box to search a linked contract. Click the search icon to open Search for Contract pop-up.

  5. Following search parameters are available in the pop-up.

    • Artist

    • Contract ID

    • Local Contract Reference Number

    • Contracting Party

    • Clearance Admin Company

  1. Select or enter relevant information and search for a linked contract

  2. Select a linked contract, the pop-up is closed and selected contract is displayed in the field. The linked contract name includes Artist name, Clearance admin company, and commencement date separated by comma.

  3. You can enter or select relevant information as search criteria from the Resource parameters section.

  4. Also, you can specify Track’s Rights parameters as search criteria by entering or selecting information for the fields in the Tracks Rights Parameters section.

Track Rights Parameters

  1. Territorial Rights – Click the Select/Remove Countries hyper link to open the Select/Remove Countries pop-up to select one or more countries for search.

  2. The pop-up displays list of countries excluding territories. You can select one or more countries by clicking the check box adjacent to countries you wish to select.

  3. The selected countries are displayed in the ‘Selected Countries’ section adjacent to the countries list.

  4. You can remove the selected countries by clicking the corresponding [x] icon in the Remove All section.

  5. After selecting one or more countries and clicking on OK, pop-up gets closed and the selected countries are shown as comma separated in the Territorial Rights text box.

Note:If user selects multiple countries and clicks on search then those applicable repertoire items should be displayed which have rights to be exploited in ALL countries selected by user. For example: If user selects France & Italy, it should return rows where they have rights to be exploited in both France AND Italy.

  1. Rights Review Status - Select a status from the drop-down.

  2. Digital Exploitation Rights - Select ‘Yes or ‘No’ from the drop-down.

  3. Digital Unbundling Allowed - Select ‘Yes or ‘No’ from the drop-down

Note:If user selects multiple countries and clicks on search then those applicable repertoire items should be displayed which have rights to be exploited in ALL countries selected by user. For example: If user selects France & Italy, it should return rows where they have rights to be exploited in both France AND Italy.

  1. Click Search. The Tracks Rights repertoire results that match the search criteria specified are displayed as list in the Search Results grid. The search Results grid is displayed as Paged format.

  2. The Search Results section include the following fields:

    • Traits

    • ISRC

    • UPC

    • Artist

    • Title

    • Version Title

    • Release Date - Earliest release date across all releases which resource is associated to

    • Clearance Admin Company

    • Sample Exists

    • Side Artist Exists

    • P Year

    • Review Status

    • Territorial Rights

      • If the length of countries exceeds those maximum characters, then you can point the cursor on the country to view the complete TRs in a tool tip.

    • Physical Exploitation Rights

    • Digital Exploitation Rights

    • Digital Unbundling Allowed

  3. You select one or more rows from the search results and clicks on Review Rights Button then system ports the selected item (s) to the Custom Resource/Tracks Rights Review Work Queue.

  4. Search results can be exported to an Excel document by clicking the  button.
