Email Notification Recipient groups are managed within Address Books. Address books are assigned to contracts, so specific users can be automatically kept informed of any impending contractual key dates or tasks Creating A New Address Book

  1. Select Contract Administration -> Manage Address Book. The Manage Address Book screen is displayed. The Filter Role/Group Name grid displays the list of existing recipient groups. The grid displays the Name, Country and Email IDs of each group.


      

  1. Click Create New Entry. The Add Role/Group pop-up is displayed.


      

3. Role/Group Name - Enter a unique name in the Role/Group Name text box.

  1. Country - Specify the country or countries by searching and selecting in the Auto Suggest look-up box.

  2. Email IDs - Enter all Email addresses of the group members, separated by a comma or semi-colon.

  3. Click Save. The Add Role/Group pop-up is closed and the newly added group is listed in the grid.

Editing Address Books

  1. Select Contract Administration -> Manage Address Book, then select the check box corresponding to the recipient group to edit.

  2. Click Edit. The Edit Role/Group pop-up is displayed.


      

  1. Edit the recipient group by entering or amending the required information.

  2. Click Save. The Edit Role/Group pop-up is closed and the saved changes are displayed in the grid.

Deleting Address Books

  1. Select Contract Administration -> Manage Address Book. , and select the check box corresponding to the recipient group to delete.

  2. Click Delete. A confirmation message is displayed.

  3. Click Yes to delete or Cancel to cancel the deletion.

  4. The deleted recipient group is removed from the grid.
