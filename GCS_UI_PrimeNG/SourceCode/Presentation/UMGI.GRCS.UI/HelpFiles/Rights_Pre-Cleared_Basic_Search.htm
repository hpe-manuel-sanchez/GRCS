Rights & Pre-Cleared Basic Search

Only users with View Repertoire Rights/View Repertoire Rights Sensitive data’  privileges can search and view the rights data created for a release/resource/track in GRS. Using the Rights Resources Basic Search, user can review the rights for required resources. You can click the following accordions to enter the respective fields for reviewing the rights of the required resources.

  • Resource Parameters

  • Resource Rights Parameters

  • Pre-Cleared

  1. Select Repertoire Rights Search -> Rights & Pre-Cleared -> Basic Search. The Rights & Pre-Cleared Basic Search screen is displayed.

  1. By default, the ‘Resources Parameters’ section is displayed. You can enter one or more combination in the search criteria for the required resources

Resources Parameters

  1. Also, you can specify Resources Rights parameters as search criteria by entering or selecting information for the fields in the Resource’s Rights Parameters section.

Resources Rights parameters

  1. You can select ‘Yes’ or’ No’ for the following Pre-Cleared Types fields.

Pre-Cleared

    • Top Price Compilation

    • Mid Price Compilation

    • Budget Compilation

    • Direct Marketing

    • Premium

    • Master/Synchronization Use

  1. Click Search. The Resources Rights & Pre-Cleared repertoire results that match the search criteria specified are displayed as list in the Search Results grid. The serach Results grid is displayed as Paged format.

  1. The Search Results section include the following fields:

    • Traits

    • ISRC

    • UPC

    • Artist

    • Title

    • Version Title

    • Release Date - Earliest release date across all releases which resource is associated to

    • Linked Contract

      • Shown as a linked contract icon in grid, with a tool tip as- Concatenation of Artist or Contracting Party if no Artist, and Contract Effective Date.

      • When the linked contract is a Split contract, then icon for split contract is displayed with a tool tip same as above.

    • Clearance Admin Company

    • P Year

    • Review Status

    • Active for Marketing

    • Territorial Rights

      • If the length of countries exceeds those maximum characters, then you can point the cursor on the country to view the complete TRs in a tool tip.

    • Rights Period

    • Lost Rights

    • Lost Rights Reason

    • Lost Rights Date

    • Physical Exploitation Rights

    • Digital Exploitation Rights

    • Mobile Exploitation Rights

    • PPB Revenue Claim

    • Digital Unbundling Allowed

    • Top Price Compilation

    • Mid Price Compilation

    • Budget Compilation

    • Direct Marketing

    • Premium

    • Master/Synchronization Use

    • Pre-Clearance Territory Exclusions - Set of pre-clearance types associated with a resource.

  1. You select one or more rows from the search results and clicks on Review Rights Button then system ports the selected item (s) to the Custom Resource Rights Review Work Queue.

  2. Search results can be exported to an Excel document by clicking the   button.
