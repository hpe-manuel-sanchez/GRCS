Rights & Digital Restrictions Resources & Tracks Basic Search

Only users with View Repertoire Rights/View Repertoire Rights Sensitive data’  privileges can search and view the rights data created for a release/resource/track in GRS. Using the Rights Tracks Basic Search, user can review the rights for required tracks. You can click the following accordions to enter the respective fields for reviewing the rights of the required Resources/Tracks:

  • Resource Parameters

  • Resource's & Track's Rights Parameter

  • Digital Restrictions

  1. Select Repertoire Rights Search -> Rights & Digital Restrictions -> Show Resources & Tracks -> Basic Search. The Rights & Digital Restriction Resources & Tracks Basic Search screen is displayed. By default, the ‘Resource Parameters’ section is displayed.

  1. By default, the ‘Resources Parameters’ section is displayed. You can enter one or more combination in the search         criteria for the required resources.

Resources Parameters

  1. ISRC – Free text field with ‘Browse’ icon. You can enter an ISRC.

  1. Also, you can browse and upload an ISRC by clicking the   icon.

  1. Note:If you provide an ISRC, all other search fields in this section are disabled.

  2. Resource/Track Title – Free text field to enter the resource title.

  3. Resource Type – Select a resource from the drop-down.

  4. Artist Name – Free text field to enter artist name and search.

  5. Exact Search – Select this check box to set the Artist Name field as exact match search field.

  6. Linked Contract Name – Multi line free text box to search a linked contract. Click the search icon to open Search for Contract pop-up.

  7. Following search parameters are available in the pop-up.

    • Artist

    • Contract ID

    • Local Contract Reference Number

    • Contracting Party

    • Clearance Admin Company

  1. Select or enter relevant information and search for a linked contract.

  2. Select a linked contract, the pop-up is closed and selected contract is displayed in the field. The linked contract name includes Artist name, clearance admin company, and commencement date separated by comma.

  3. Repertoire linked to multiple contracts – If you provide one or more linked contracts, then this check box is disabled.

  4. Also, you can specify Resource’s and Track’s Rights parameters as search criteria by entering or selecting information for the fields in the Resource’s & Track’s Rights Parameters section.

Resource’s & Track’s Rights Parameters

 15.  Territorial Rights – Click the Select/Remove Countries hyper link to open the Select/Remove Countries pop-up to select one or more countries for search.

  1. The pop-up displays list of countries excluding territories. You can select one or more countries by clicking the check box adjacent to countries you wish to select.

  2. The selected countries are displayed in the ‘Selected Countries’ section adjacent to the countries list.

  3. You can remove the selected countries by clicking the corresponding [x] icon in the Remove All section.

  4. After selecting one or more countries and clicking on OK, pop-up gets closed and the selected countries are shown as comma separated in the Territorial Rights text box.

Note:If user selects multiple countries and clicks on search then those applicable repertoire items should be displayed which have rights to be exploited in ALL countries selected by user. For example: If user selects France & Italy, it should return rows where they have rights to be exploited in both France AND Italy.

  1. Rights Review Status - Select a status from the drop-down.

  2. Rights Period - Select a period from the drop-down.

  3. Lost Rights Indicator - Select ‘Yes or ‘No’ from the drop-down.

  4. Physical Rights Exploitation - Select ‘Yes or ‘No’ from the drop-down.

  5. Lost Rights Reason – Select a reason from the drop-down.

Note:You can select a lost rights reason and view it in the Search Results section irrespective of your privileges, that is, all the users of GRS can select and view the lost rights reason.

  1. Digital Exploitation Rights - Select ‘Yes or ‘No’ from the drop-down.

  2. Digital Unbundling Allowed - Select ‘Yes or ‘No’ from the drop-down.

Note:If you have selected ‘No’ in the Digital Exploitation Rights dropdown, then this filed is disabled.

  1. Mobile Exploitation Required - Select ‘Yes or ‘No’ from the drop-down.

  2. PPB Revenue Claim - Select ‘Yes or ‘No’ from the drop-down.

  3. Active for Marketing - Select ‘Yes or ‘No’ from the drop-down.

  4. Lost Rights Date - Select from date and to date from the respective calendar lookups.

  5. You can enter or select relevant information as search criteria in the Digital Restrictions section.

Digital Restrictions

  1. Use Type – Select a use type from the drop-down

  2. Commercial Models – Select a option from the dropdown

  3. Restriction Exists – Select this check box if digital restrictions are available.

  4. Restrictions Doesn’t Exists - Select this check box if digital restrictions are not available.

  5. To add an individual Digital Restriction row, click the Add Restrictions hyper link. The Digital Restriction row is displayed.

  6. To remove a Digital Restriction row, click the Remove icon corresponding to the row you wish to remove.

  7. Click Search. The Tracks Rights & Digital Restrictions repertoire results that match the search criteria specified are displayed as list in the Search Results grid. The search Results grid is displayed as Paged format.

Note:If you click Reset, all the search criteria entered will be cleared, ready for entering new search criteria.

  1. The Search Results section include the following fields:

    • Traits

    • ISRC

    • UPC

    • Artist

    • Title

    • Version Title

    • Release Date - Earliest release date across all releases which resource is associated to

    • Linked Contract

      • When the linked contract is a Split contract, then icon for split contract is displayed with a tool tip same as above.

    • Clearance Admin Company

    • P Year

    • Review Status

    • Active for Marketing

    • Territorial Rights

      • If the length of countries exceeds those maximum characters, then you can point the cursor on the country to view the complete TRs in a tool tip.
