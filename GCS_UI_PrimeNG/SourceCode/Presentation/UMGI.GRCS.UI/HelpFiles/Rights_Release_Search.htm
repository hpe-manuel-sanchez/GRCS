Rights Release Search

Only users with View Repertoire Rights/View Repertoire Rights Sensitive data’  privileges can search and view the rights data created for a release/resource/track in GRS. Using the Rights Resources & Tracks Search Using Release Parameters, user can review the rights for required Resources and Tracks. You can click the following accordions to enter the respective fields for reviewing the rights of the required Resources/Tracks:

  • Release Parameters

  • Release Rights Parameters

  1. Select Repertoire Rights Search -> Rights -> Show Releases. The Rights Releases screen is displayed. By default, the ‘Release Parameters’ section is displayed.

Release Parameters

  1. UPC – Free text field with ‘Browse’ icon. You can enter an UPC. Also, you can browse and upload UPCs by clicking the  icon

Note:If you provide an ISRC or upload a list of UPCs, all other search fields in this section are disabled.

  1. Release Title – Free text field to enter the release title.

  2. Artist Name – Free text field to enter artist name and search.

  3. Exact Search – Select this check box to set the Artist Name field as exact match search field.

  4. Linked Contract Name – Multi line free text box to search a linked contract. Click the search icon to open Search for Contract pop-up.

  5. Following search parameters are available in the pop-up.

    • Artist

    • Contract ID

    • Local Contract Reference Number

    • Contracting Party

    • Clearance Admin Company

  1. Select a linked contract, the pop-up is closed and selected contract is displayed in the field. The linked contract name includes Artist name, Clearance admin company, and commencement date separated by comma.

 

Release Rights Parameters

 

  1. Exploitation Countries – Click the Select/Remove Countries hyperlink to open the Select/Remove Countries pop-up to select one or more countries for search.

  2. The pop-up displays list of countries excluding territories. You can select one or more countries by clicking the check box adjacent to countries you wish to select.

  3. The selected countries are displayed in the ‘Selected Countries’ section adjacent to the countries list.

  4. You can remove the selected countries by clicking the corresponding [x] icon in the Remove All section.

  5. After selecting one or more countries and clicking on OK, pop-up gets closed and the selected countries are shown as comma separated in the Territorial Rights text box.

  6. Rights Period - Set the contractual Rights Period using the available drop-down values. The Rights Period is set to 'Empty' by default.

  7. Lost Rights Indicator - Select ‘Yes or ‘No’ from the drop-down.

  8. Physical Exploitation Rights - Select ‘Yes or ‘No’ from the drop-down.

  9. Lost Rights Reason – Set the Lost Rights Reason using the available drop-down values.

  10. Digital Exploitation Rights - Select ‘Yes or ‘No’ from the drop-down.

  11. Lost Rights Date - Select from date and to date from the respective calendar lookups.

  12. Click Search. The Releases Rights repertoire results that match the search criteria specified are displayed as list in the Search Results grid. The search Results grid is displayed as Paged format

 

  1. The Search Results section include the following fields:

    • Traits

    • UPC

    • Artist

    • Title

    • Version Title

    • Release Date- Earliest release date across all releases which resource is associated to

    • Linked Contract

      1. Shown as a linked contract icon in grid, with a tool tip as- Concatenation of Artist or Contracting Party if no Artist, and Contract Effective Date.

      2. When the linked contract is a Split contract, then icon for split contract is displayed with a tool tip same as above.

    • Clearance/Data Admin Company

    • P Year

    • Review Status

    • Territorial Rights

      1. If the length of countries exceeds those maximum characters, then you can point the cursor on the country to view the complete TRs in a tool tip.

    • Rights Period

    • Lost Rights

    • Lost Rights Reason

    • Lost Rights Date

    • Physical Exploitation Rights

    • Digital Exploitation Rights

    • Rights Data Source

  1. You select one or more rows from the search results and clicks on Review Rights Button then system ports the selected item (s) to the Custom Release Rights Review Work Queue.

  2. Search results can be exported to an Excel document by clicking the  button.
