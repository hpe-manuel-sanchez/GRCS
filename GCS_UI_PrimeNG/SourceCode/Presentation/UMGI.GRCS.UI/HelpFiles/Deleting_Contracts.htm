Deleting Contracts

After searching for a contract, you can delete a contract.

  1. Search for the required contract(s) in the Search Contract screen.

  2. From the search result grid, select the check box corresponding to the contract row to delete.

  3. Click Delete Contract. A confirmation message to delete the contract is displayed.

  4. Click Ok to delete.

  5. The deleted contract and its rights data is removed from the list of contracts displayed in the search result grid and permanently removed from GRS.

Notes:

  1. If there one or more child contracts linked to the selected contract, then a message is displayed stating that the selected contract cannot be deleted as child contract(s) are linked to it.

  2. If there is any repertoire linked to the selected contract, then a message is displayed stating that the selected contract cannot be deleted as repertoire is linked to it.

  3. Only users with specific rights can delete contracts. Contact your local Super User for more information on user rights.
