Searching for Contracts

You can search for all existing contracts under the Search For Contracts section. From the search results, you can perform tasks such as:

  • Copy Contracts

  • Open Contract

  • Delete contracts

Searching for Contracts

  1. Select Contract Administration -> Search Contract. The Search Contract screen is displayed.  This screen displays the primary search fields of ‘Artist’, ‘Workflow Status’, and ‘Contract Status’. You can enter one or a combination of criteria to search for a contract.


     

  1. If you wish to apply advanced search criteria, click the Advanced Search header. The advanced search criteria fields are then displayed.


     

  1. Enter the required fields and click Search. A list of contracts that match the search criteria is displayed in the search result grid of the screen.


     

Note: If you click Reset, all the search criteria entered will be cleared, ready for entering new search criteria.

  1. Each row in the Search Grid displays the basic contract details including Artist Name, Workflow Status, and Clearance Admin Company.

Note: If a contract is a parent contract, an indicator ‘P’ is displayed corresponding in that contract row. Similarly, if it is child contract, then the indicator displayed is ‘C’. For an individual contract, there will be no indicator.  

  1. The list of contracts in the search result grid can be sorted in ascending or descending alphanumeric order by clicking each column header.

  2. Search results can be exported to an Excel document by clicking the button.
