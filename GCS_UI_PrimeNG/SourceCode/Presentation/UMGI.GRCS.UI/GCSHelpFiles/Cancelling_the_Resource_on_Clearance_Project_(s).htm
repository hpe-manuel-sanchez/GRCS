Cancelling the Resource on Clearance Project (s)

1. Login to GCS and the home page of the system is displayed.

2. Request which should be cancelled can be accessed in two ways

  •  From the Requestor's Inbox, click anywhere on the project except the project ref no. and system opens the RHS of the selected project.

  • From the requestor's inbox, click on the project ref no. hyperlink and navigate to the request summary tab.

Note: System does not enable the cancel button for resources of existing release added to Regular/Non Traditional Project.

3. Click Cancel button against the relevant resource to cancel the request.

4. System updates the status of resource as cancelled.

5. An email notification is sent to the reviewer informing about cancellation of the request and the request is removed from the reviewer’s inbox.

Note: Cancel is only applicable to resource having waiting status.
