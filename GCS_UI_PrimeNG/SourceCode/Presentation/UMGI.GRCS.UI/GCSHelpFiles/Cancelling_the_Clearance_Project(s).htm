Cancelling the Clearance Project (s)

* Cancellation can only happen for Submitted/Resubmitted/Re-Open Project Status

1. Login to GCS and the home page of the system is displayed.

2. Navigate to the Requestor Inbox and click on the Project Reference Number to cancel the project.

3. Click Cancel and system sets the status of the project to be ‘cancelled’.

Note:1. System updates the status of pending request as cancelled.
2. System doesn’t update the status of the requests which are approved/conditionally approved/rejected/cancelled.
 

4. All the fields of the cancelled project are set as non-editable.

5. System sends email notification to the reviewer about project cancellation for waiting resources and removes the waiting resources from reviewer’s inbox.

6. The clearance system sends an email to all the users in reviewer workgroup which had finally (conditionally) approved the request.

Note:

1. If the request was auto approved, System does not send the email to users.

2. You can also cancel the project from requestor inbox RHS project actions by directly clicking on “mark as cancel” button.
