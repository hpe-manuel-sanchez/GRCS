Audit Trails and History

1. Login to GCS and the home page of the system is displayed.

2. System displays a history-link to view the history on the screen wherever it is applicable.

Note: Location of history link on each screen is described in the points below.

3. Click on History and clearance system displays a pop up with different fields listed on the left hand side grouped by sections.

4. Left Hand Section:  The left hand side section displays the list of fields are sorted alphabetically. The fields are grouped under various sections wherever possible.

5. View History: You can select one or more fields on the left hand side section and click on view history. System displays the history of the selected fields in different tables on the right hand side section of the audit trail screen.

6. Reset Selection: Click on Reset Selection, to clear the selection of input fields. The right hand side will not be reset until the user selects some other fields and clicks on view history again.

7. Right Hand Section: The right hand side details in the table should be sorted based on Date Time with latest one on top. It displays the Table Name and Table column for the fields which you chose to view the history.

8. If there is no data to be displayed for the selected field, the clearance system displays a blank table.

9. Export to Excel: You can click on export to excel to view the details visible on right hand side section of the screen in a excel file. The excel should populate all the tables which are displayed on the screen

10. The history link is available at different locations on various screens. Please find the details of history link of every screen in GCS application.
