Amending Parent Workgroups

* Only RCC Admin or workgroup users with manage workgroup rights can amend the existing workgroup.

Fields marked with an asterisk * are mandatory fields

1. Login to GCS and the home page of the system is displayed.

2. Select Workgroup -> Manage Workgroup. Click the Manage Workgroup tab.

3. The Maintain Workgroup screen with search options is displayed.

4. You can search the workgroup through the available search by options, based on either one or combination of the search criteria:

  • Search by workgroup name

  • Search by role in drop-down

  • Search by company name and/or ISAC Company code

  • Search by user name

  • Search by Territory/Country name

5. Enter the search criteria and click Search. The workgroups that match the search criteria are displayed as list in an alphabetical order of workgroup name.

6. You can choose a workgroup from the list to amend the workgroup and click the Workgroup Name hyperlink. You are navigated to the Maintain Workgroup screen with the workgroup details.

7. You can amend the name of the work group.

8. All the fields are in editable mode except the Role field.

8. To amend company (s) of the parent workgroup, click the Manage Company button. The Manage Company pop-up is displayed. For Description: Please Refer Manage Company.

Note: After navigating to manage company popup, by default system will display previously added company(s).

9. To amend territory (s) of the parent workgroup, click the Manage Territories button. The Manage Territory pop-up is displayed. For Description: Please Refer Manage Territories.

10. To amend User (s) of the parent workgroup, click the Manage Users button. The Manage Users pop-up is displayed. For Description: Please Refer Manage Users.

Note: After navigating to manage user popup, by default system will display previously added user(s).

11. You can amend the R2 contact field.

12. Click Save. The workgroup details are updated, if there is no uniqueness violation due to updated fields.
