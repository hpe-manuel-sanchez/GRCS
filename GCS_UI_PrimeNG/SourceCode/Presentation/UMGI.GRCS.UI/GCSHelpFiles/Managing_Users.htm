Managing Users

*RCC Admin/User with manage workgroup rights can Add/Remove User to Workgroups.

1. Login to GCS and the home page of the system is displayed.

2. Select Workgroup -> Manage Users. Click the Manage users tab.   

3. The clearance system displays manage users of a workgroup screen.

4. You can search users based on the following search criteria:

  • Search by User Name

  • Search by User ID

  • Search by Country

5. You can enter text in the search criteria fields and click Search. The system displays a list of users based on the above search criteria in the pop-up in an alphabetical order.

Note:  If you click Reset, all the search criteria entered will be cleared, ready for entering new search criteria.

6. You can perform two tasks on Manage User screen:

  • Add User(s) to workgroup(s)

  • Remove User(s) from workgroup(s)

Add User(s) to Workgroup(s)

7. Set the flag corresponding to the user and click on Add Button. You can select only one user at a time.

8. System displays the Add User to Workgroup pop up.

9. You can search for a workgroup based on one or more of the following search criteria

  • Search by Name

  • Search by Role

  • Search by Company

  • Search by user

  • Search by Territory/Country

10. By clicking on Search, system displays the search results for desired search criteria as shown below:

11. System displays all the search results matching the search criteria.

12. You can set the flag to select one or more workgroups, set the user as default for one or more workgroups and assign rights type to the selected user in one or more workgroup and click on Add user button.

Notes:

1. The default flag will only be enabled if role of the workgroup is reviewer.

2. By default, system sets the flag on the rights type based on the role of the work group.

3. You can select/deselect any of the rights against the workgroup.

13. System adds the user to selected workgroups and navigates back to Add/Remove Users from Workgroup(s) screen with the same search results based on previous search criteria.

Remove User(s) from Workgroup(s)

14. Set the flag corresponding to the user and click on Remove Button. You can select only one user at a time.

15. System displays the Add User to Workgroup pop up as displayed below:

16. System displays all the active workgroups with which the user is linked with the following workgroup details.

  • Workgroup Name

  • Status Type

  • Role Name

  • Company

  • Default User(s)

  • Countries

17. You can select one/more/all workgroups from which the user should be removed and click on Remove button.

18. System removes the user (from the user and the default user fields) from the selected workgroup(s) and navigates back to manage users of a workgroup screen with the previous search results displayed.

Note: If the user being removed is the last user on the workgroup, the clearance system does not remove the user from that workgroup and displays a message “<<name of user>> is the last user linked to the <<workgroup name>> <<role>>. Please add another user to the workgroup to remove the current user.
