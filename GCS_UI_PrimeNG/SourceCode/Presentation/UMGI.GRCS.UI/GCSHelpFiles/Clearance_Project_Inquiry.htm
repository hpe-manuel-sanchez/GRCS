Clearance Project Inquiry

* User having inquiry role with inquiry rights or requestor or reviewer or RCC Admin can use this functionality

*It ensures to view the project details in Read only mode

1. Login to GCS and the home page of the system is displayed.

2. Navigate to the Projects>>Projects Inquiry as displayed below:

3. Search Project for Project Inquiry screen is displayed with two sections

4. You can enter one or more relevant search criteria.

Note: By Clicking on Reset, system reset the entered field.

5. Click Search and system displays the search result as shown below:

Note:  If the projects available in Clearance system based on search criteria do not match with the company/country/territory rights that user has access, no results are displayed in the screen.

6. On clicking Project Reference hyper link, the separate browser/tab is opened with the project details of the selected Project Reference Number appearing on the Browser Tab title

Note:All the fields will be in Read only mode
