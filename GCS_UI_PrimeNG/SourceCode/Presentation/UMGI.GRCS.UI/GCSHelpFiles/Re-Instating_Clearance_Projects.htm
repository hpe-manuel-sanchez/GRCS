Re-Instating Clearance Projects

1. Login to GCS and the home page of the system is displayed.

2. Navigate to manage cancelled/completed project and search for the relevant project.

3. System displays the Search Cancelled/Completed Projects.

4.  Search for the project based on defined search criteria and click on Project Reference Number to open the project in new browser/tab.

5. System displays the Reinstate button at the bottom of the screen.

Note: All the fields are set as non-editable.

6. Click Reinstate and System sets the project status to its previous status before it was cancelled.

7. The status of all the requests on summary tab is changed to its state prior to project cancellation.

8. System sends email and inbox notification to the reviewer once the waiting requests are re-instated in the inbox.

9. All project fields become editable as and when the project is re-instated.
