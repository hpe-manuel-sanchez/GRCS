Modifying Clearance Projects

* User in requestor workgroup and having requestor rights can modify clearance project in Clearance System.

1. Login to GCS and the home page of the system is displayed.

2. Navigate to the Requestor Inbox and click on the Project Reference Number to view and edit the fields of the projects.

3. You can edit any of the fields of the project and click on Save.

Note:

1. Save for unsubmitted will just save the changes and there will be submit button to submit the project to trigger routing.

2. For already submitted project, the status will change to re-submitted if the changed fields trigger re-submission of all the requests on the project.
