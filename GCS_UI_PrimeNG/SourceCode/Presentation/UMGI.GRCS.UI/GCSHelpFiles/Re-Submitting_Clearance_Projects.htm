Re-Submitting Clearance Projects

* Re-Submission happens for Submitted/Resubmitted/Re-Opened Project(s)

1. Login to GCS and the home page of the system is displayed.

2. Navigate to the Requestor Inbox and click on the Project Reference Number to view and edit the fields of the projects.

Note: You can open any project with status type Submitted/re-submitted/re-opened.

3. On mouse hovering on the fields which triggers re submission is displayed as shown below:

4. Once you are done with the amendments, click on SAVE.

5. System displays the following message:

Note:1. If you click on No, system discards all the changes.
2. If you click on YES, system checks for cancelled or rejected resource (s) in the selected project.

6. System displays the below message for resubmission of cancelled/rejected resource(s).

7. If you click Yes, the pop up is displayed containing the list of cancelled or rejected resources

.

Note: If you click on No, then system resubmits only approved/ conditional approved and pending resources

8. Select one or more resources and click on Submit.

9. After Submitting, the project status is set to Re-Submitted and the reason for re-submission will be available against each resource in the routing details pop up.
