Include Resource (s)

*Include Resource (s) is enabled only for resource (s) in Master Projects.

1. Login to GCS and the home page of the system is displayed.

2. Request which should have include action can be accessed via two ways:

  • From the Requestor's Inbox, click anywhere on the project except the project ref no. and system opens the RHS of the project which has review status as Excluded.

  • From the requestor's inbox, click on the project ref no. hyperlink and navigate to the request summary tab.

3. System displays the include button in the request summary tab for all the resources which have the status as Excluded.

4. By clicking on Include button, system changes the status of the resources as approved or conditionally approved.

5. System sends e-mail notification in the inbox of the reviewer regarding the change of status of resource(s).
