Mimic User

* The action on Mimic User can only be performed by RCC Admin

1. Login to GCS and the home page of the system is displayed.

2. Select Admin -> Mimic User. Click on Mimic User button.

3. Mimic User pop up is displayed wherein you search by below criteria

4. Enter the relevant Search criteria and click Search.

Note: By clicking Reset, you can clear the entered fields

5. System displays the search result for the entered search criteria

6. Select the user and click Mimic User.

Note:By Clicking Cancel, you navigate back to RCC Admin home page

7. The home page of the selected user with all the other navigations that the selected user has access is displayed.

8. Exit: By clicking Exit link on the selected user’s screen, you navigate back to RCC Admin home page.
