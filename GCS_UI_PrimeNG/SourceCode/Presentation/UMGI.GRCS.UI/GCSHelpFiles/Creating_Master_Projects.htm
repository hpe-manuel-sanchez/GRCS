Creating Master Projects

* User with requestor role and having requestor rights can create a master project in Clearance System.

Fields marked with an asterisk * are mandatory field

Project reference no. is only created after Save or Submit

User can skip between the tabs without saving

It is advised however, that users save regularly since there is not Auto save function so in case of system issues data will be lost when not saved.

1. Login to GCS and the home page of the system is displayed.

2. Select Projects -> Create Projects -> Master Project. Click on Master Project Tab

3. By default, system opens the project information tab with the following sections:

  • Project Information

  • Request Type

Project Information

4. By default, System displays the following fields on the Project Information tab

  • Status of the project as UnSubmitted.

  • Created date  as today’s date

  • Requestor name

5. Project Title: Text field for entering the Project Title.

6. Local Reference Number: Enter as Required

7. Details: You can enter the details related to project.

8. Release Territories: To add Territory/Country to the project, Refer Manage Territories in managing workgroup.

9. You can view the countries in each territory on the tool tip by mouse hovering over the territory as displayed in the above screen grab.

10. Requesting Company:  By default, system displays the first company linked with the Requestor’s Work Group.

11. If there is more than one company linked with the Requestor’s work group or if the Requestor is a part of more than one requestor workgroup, system displays the default company based on your settings in User Preferences. You can select any other company which are linked with the Workgroup(s) from the requesting company drop down.

12. You can select any other company which are linked with the Workgroup(s) from the requesting company drop down.

13. Upload Project Documents: You can add related project documents by clicking on Browse button.

14. You cannot upload project documents which exceeds the size limit of 4MB, system displays the error message “Attached document exceed 4MB size limit”.

15. You can select multiple documents from same/ different folder one by one but total number of documents uploaded should not exceed 10. The maximum size of all the documents cannot exceed 4 MB limit.

16. Once selection of documents is done, you can upload the project documents by clicking on Upload button.

17. By clicking on Remove link, you can remove the uploaded documents.

18. Sensitive exploitation: Select if the project is sensitive exploitation.

19. One Stop – Select for One-Stop Routing

20. If you select one stop, then you have to enter one stop reason comments as a text.

21. Currency: By default, system displays the default currency based on your settings in User Preferences.

22. You can select any other currency from the drop down list, you  the currency and can edit the currency from any of the other tabs.

Request Type Section

23. Client Name: Enter the Client Name related to project as text .

24. Client Website: You can enter in the Client Website field as free text.

25. License Term: Enter the License Term related to project as a free text.

26. Select at least one option from Advertising, Film, Trailer and other.

Notes:

1. You cannot select combination of Advertising, Film and Trailer

2. You can only combine Others options with Advertising, Film and Trailer.

 

27. Advertising Section: If the flag is set on Advertising field, then system displays the following fields for you to select.

28. Advertised Products: Enter the products used for advertising as free text in Advertised Products fields.

29. Media: You can set the flag to select one or more options from TV, Cinema, Video, Radio, Internet, Others.

Note: * Selecting at least one option is mandatory
 

30. Initial Number of Videos: If you set the flag to select video, then system displays the “initial number of videos” field to enter the details as number.

31. Comments: If you set the flag to select Other, then you have to enter the comments as text in comments field.

32. Number of Spots: You can enter number of spots as number maximum up to 7 digits.

33. Duration From and Duration To: You can select a date from/to which this request is applicable from the calendar control in “Duration From”/ “Duration to” field.

34. Optional Additional Rights: You can enter optional additional rights as free text.

35. Film Section: If the flag is set on Film field, then system displays the following media options for you to select.

Note * Selecting at least one option is mandatory.
 

36. Initial Number of Videos: If you set the flag to select video, then system displays the “initial number of videos” field to enter the details as number

37. Comments: If you set the flag to select Other, then you have to enter the comments as Free text in comments field

38. Optional Additional Rights:  You can enter optional additional rights as free text.

39. Trailer Section:  If the project includes a Trailer request, then you should select on Trailer check box and system displays the options for TV, cinema, video, internet, videos and Other.

Note:* Selecting at least one option is mandatory

40. Initial Number of Videos: If you set the flag to select video, then system displays the “initial number of videos” field to enter the details as number

41. Comments: If you set the flag to select Other, then you have to enter the comments as Free text in comments field

42. Optional Additional Rights: You can enter optional additional rights as free text.

43. Other Section: If the project includes a Other request, then you should select on other check box and system displays the options for TV, cinema, radio, video, internet and other.

Note: * Selecting at least one option is mandatory
 

44. Initial Number of Videos: If you set the flag to select video, then system displays the “initial number of videos” field to enter the details as number.

45. Comments: If you set the flag to select Other, then you have to enter the comments as Free text in comments field.

46. Optional Additional Rights: You can enter optional additional rights as text.

47. After entering the Project information tab fields, you navigate to the Resources tab in create project screen.

Resources Tab

48. Search for Resources: Enter any one combination of following fields to find and add required resource(s) to the project and click Search.

  • ISRC

  • Resource Title

  • Artist Name

  • UPC

49. System can identify below search results based on the search criteria

  • Unique Search Result

  • Multiple Search Result

  • No search results

Unique Search Result

50. For the unique search result, the resource search based on above mentioned search criteria displays the selected resource as shown in the below screen. Note: mobile resources are excluded from the search/results

Notes:

1. Mobile resources are excluded from the search/results

2. The system highlights those resources where rights type is non-UMG and/or non-exclusive license.

Multiple Search Results

51. System displays the below screen with relevant search results based on the search criteria entered on resource tab.

Notes:

1. You can refine the search by adding more search criteria like Artist ID, UPC, version title and resource type.

2. By default, the resource type displayed on the screen is Audio and you can change it to video.

3. Include Mobile: By default Mobile resources are excluded from the search. If you want to search for Mobile resources you will need to tick the ‘Include Mobile’ option

4. You are not allowed to add duplicate resources in the project.

Note: By clicking on Reset button, you can clear the search options entered to search for resource.

52. You can click on the indicator arrow next to ISRC to view the rights information of the resource.

Notes:

1. If the resource rights are available in GRS, system displays the resource rights set information.

2. If the resource rights are not available in GRS, then system displays the artist contract details in the rights information section.

3. If there is no contract available a message ‘no data available’ will be shown

53. You set the flag to select one or more resources and click on Add to Project.

Add Freehand Resource

54. If no resources are available based on the search criteria a message ‘no data available’ will be shown and system enables the Add Free hand button.

55. System displays the below screen with list of fields to be entered :

56. Add Artist on Freehand Resource: You click on Manage Artist button to link the artist(s) with free hand resource. System opens manage artist window in the popup.

57. You can search for artist based on one or more combination of search criteria.

  • Search by Artist Name

  • Search by Artist ID

Note: You can click on Reset Button to clear the input in the search criteria.

58. System displays the search results with the following fields:

59. You select the artist and click on Add, the system displays the selected artist(s) on the Search for resources screen.

Note:1. Adding artist is mandatory
2. If you want to add more than one artist with free hand resource, click on manage artist again.
 

60. In order to remove the linked artist, click on Manage Artist button. Select the artist you wish to remove and Click on Remove Button, system removes the already linked artist(s).

61. Freehand Resource Title: Enter the name of the free hand resource as text in Title field.

62. Freehand Version Title: You can add the Version title as text.

63. Freehand Recording Type: By default system shows the default value as studio in Recording type

Note: You can select the recording type from the drop down.
 

64. Freehand Resource Type: You can select the resource type value from the drop down.

Note: By default, system displays the resource type value as Audio.

65. Freehand Music Type: You can select the music type from drop down. By default, system shows the pop as selected value.

66. Freehand Resource Time: You can enter the resource time for free hand resource in duration field.

Note: It should be in hh:mm:ss format.
 

67. Freehand Excerpt Time: Enter excerpt time for free hand resource as hh:mm:ss

Note: *It is a mandatory field.
 

68. Suggested Fee: You can enter the suggested fee for the free hand resource as currency.

69. Comments: You can enter the comments as free text.

70. Manage Release Countries: To add Territories/countries, click on Manage Territories. System will display Manage Territory popup. For Description: Please Refer Manage Territories.

Notes:

1. System will automatically link the project include/exclude territories /countries with the resource at the back end.

2. You cannot add any territory/country; you can only exclude one or more of the territories/countries listed under Included Territories in the project information tab

3. You must enter these in the Excluded Territories on the Resource

4. The territories/countries listed under Excluded Territories in the project information tab cannot be entered in the included Territories on the resource tab

71. After entering all the mandatory fields, you click on the Add to Project.

72. System displays the below screen with non-editable /editable fields for each resource.

Note: All the fields for free hand resource are editable.

73. Suggested Fee: Enter the suggested fee as currency.

74. Sensitive Exploitation: You can set the flag to select sensitive exploitation for any of the resources.

Notes:

1. This field is editable for each resource

2. If you set the flag on sensitive exploitation in project information tab, then system sets the flag to select sensitive exploitation for each resource on the resources tab and then you cannot edit the sensitive exploitation flag for each resource.

3. If you have not set the flag to select sensitive exploitation on the project information tab, then clearance system does not set the flag to select sensitive exploitation for any of the resources and you can set the flag to select sensitive exploitation for any of the resources.

75. Comments: You can enter the text in comments field, if required.

76. Duplicate Resources: If the selected resource(s) is already added to the project, system discards the duplicate resource(s) and adds the other selected resource(s) to the project.

77. Save: You can click on Save button to save the project at any point in time. When Project is successfully saved, system will populate the project reference number on the project information tab.

78. Submit Project: Click on Submit, system validates for all the mandatory fields and navigates to Request Summary tab to display the current status of all the requests.

Notes:

1. Summary tab should be manually refreshed by click on Refresh on Request summary tab whenever you open the project.

2. You can submit the project directly even without having saved it, the submit function does also save the project

Request Summary Tab

79. You are navigated to request summary tab, once the project is successfully submitted to display the current status of all the requests.

80. Routing Details: You can click on Routing Details hyperlink to check the detailed history of routing as well as future steps as displayed below.

Notes:

1. Steps which are yet to be completed are highlighted in a different color code.

2. Request is displayed in the order of the routing steps with the oldest step in routing at the top.
