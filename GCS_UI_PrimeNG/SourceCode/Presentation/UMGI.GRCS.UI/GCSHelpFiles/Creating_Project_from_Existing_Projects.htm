Creating Project from Existing Projects

1. Login to GCS and the home page of the system is displayed.

2. Select Projects -> Create Projects -> Create from Existing Project .Click on Create from Existing Project Tab.

3. System displays Create New from existing project screen as shown below:

4. You can search for an existing project based on one or more combination of following criteria.

  • Project Reference Number

  • Project Type

  • 3rd Party Company

  • Project Title

  • Request Type

  • Requestor

  • Local Reference

  • Requesting Company

5. Click Search. System displays the list of projects in numerical/alphabetical order of project reference number.

Notes:

1. By clicking on Reset, system clears the entered search criteria.

2. System will display only projects for those workgroup for which the requestor has rights.

3. Set the flag to select the relevant project and click Copy.

Note: You can select only one project at a time.

4. System verifies the project type of the selected project

  • For ‘Master’ Project Type

  • For ‘Regular/Non-Traditional’ Project type

Master Project Type

5. Navigate to the Master Project Screen.

6. System copies all details on project information tab and resources tab except the following.

Notes:

1. System does not copy the resources which have cancelled status.

2. Freehand Resources for which ISRC is not found, is not copied to the project.

7. To edit the project details, add/remove resources, save and submit functionality; please refer Creating Master Project section.

Regular/Non-Traditional Project Type

8. Navigate to the Regular/Non Traditional Project Screen.

9. System will only copy the details on project information and request type screens of the existing project.

Note: System will not copy any Release and Resource information of the project.

10. To edit the project details, add/remove resources/releases, save and submit functionality; please refer Creating Regular/Non Traditional Project Section.
