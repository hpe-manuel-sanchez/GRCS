Reminder

* Remind Button is applicable only for the resources whose status is waiting

1. Login to GCS and the home page of the system is displayed.

2. Request which should have remind action can be accessed via two ways:

  • From the Requestor's Inbox, click anywhere on the project except the project ref no. and system opens the RHS of the project which has review status as waiting.

  • From the requestor's inbox, click on the project ref no. hyperlink which has review status as waiting and navigate to the request summary tab.

3. System displays the Remind button in the request summary tab for all the resources which have the status as waiting.

4. By clicking on Remind button, System sends e-mail notification to the default user of the relevant workgroup to remind the reviewer with whom the request is pending.
