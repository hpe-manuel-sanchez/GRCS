Managing Artist Contracts

* RCC Admin/User with manage workgroup rights can Manage Artist Contract.

1. Select Workgroup -> Manage Deviation -> Manage Artist Contract. Click on Manage Artist contract tab

2. The Search for Artist Contract screen with search options is displayed.

3. You can search artist contract based on the following search criteria:

  • Search by Artist Name

  • Search Artist ID

  • Search by Clearance Admin Company

4. You can enter search criteria and click Search. The system displays a list of artist contracts in the pop-up and is sorted alphabetically by artist name and then Clearance Admin Company.

5. System displays the contract flag icon color as “Green” if contract is linked with the Artist in GRS.

6. System displays the contract flag icon as “Red” if no contract is linked with the Artist in GRS.

7. You can click the arrow icon against the Artist to view the contract details.

8. If multiple contracts is available for the above selected Artist in GRS, the system displays the list of contract(s) of the Artist.

9. Select on the relevant contract(s) and click on Link to Workgroup button.

Note: If the selected contracts do not have the same clearance Admin company, system displays the error message.
 

10. The Link workgroup to Contract screen is displayed to link the relevant contract to workgroup.

11. By default, system displays the list of reviewer child workgroups whose company matches with the clearance Admin company of the selected contract

12. You can refine the search for workgroups based on one or more combination

  • Search by Workgroup Name

  • Search by Role

  • Search by Company Name/ ISAC code

  • Search by User

  • Search by Country / Territory Name

13. Set the flag to select one/more workgroup and click on Link to Contract.

Note: System links only those contracts which are not previously linked

14. Once contract is linked to the workgroup, System displays the following message.

15. System displays the selected contract(s) name in the artist contract deviations of all the work groups to which they are linked.
