Re- Apply Track (s)

* Re-Apply button in the request summary tab is displayed against each resource which has review status as rejected'.

1. Login to GCS and the home page of the system is displayed.

2. Request which should have re-apply action  can be accessed via two ways:

  • From the Requestor's Inbox, click anywhere on the project except the project ref no. and system opens the RHS of the selected project.

  • From the requestor's inbox, click on the project ref no. hyperlink and navigate to the request summary tab.

3. System displays the re-apply button in the request summary tab against each resource which has review status as rejected.

4. By clicking on Re-Apply button, system enables a comments section to enter.

5. System sends notification in the inbox of the reviewer who had rejected the request.

Note: If the request was auto rejected, system will send a notification to the RCC Admin for review.
