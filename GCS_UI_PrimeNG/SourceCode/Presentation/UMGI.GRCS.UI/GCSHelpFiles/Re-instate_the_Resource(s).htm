Re-instate the Resource (s)

* Re-instate button in the request summary tab is displayed against each resource which has review status as cancelled.

1. Login to GCS and the home page of the system is displayed.

2. Cancelled resource(s) can be accessed via two ways:

  • Through inbox

  • Through summary tab

3. System displays the Re-instate button in the request summary tab against each resource which has review status as cancelled.

4. By clicking on Reinstate button, system restores the status of the resource before cancellation.

5. System sends notification to the corresponding reviewer’s inbox for pending request.

6. System also sends an email to the reviewer for re-instated request to review the request.
