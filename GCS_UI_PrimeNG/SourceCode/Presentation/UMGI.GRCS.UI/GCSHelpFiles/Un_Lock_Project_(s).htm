Un Lock Project (s)

* The action on Unlock Projects can only be performed by RCC Admin

1. Login to GCS and the home page of the system is displayed.

2. Select Admin -> UnLock Projects. Click on Unlock Projects button.

3. Unlock Project screen  is displayed wherein you search for Project Reference Number:

4. Project Reference Number: Enter the relevant Project reference number and click Search.

Note: By clicking Reset, you can clear the entered fields.

5. System displays the search result for the entered search criteria:

6. Click Unlock button and system will update the status of the project as unlocked.

Note: This button is displayed against those projects for which the status is appearing as locked.
