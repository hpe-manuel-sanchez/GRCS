Managing Resource Contracts

1. Select Workgroup -> Manage Deviation -> Manage Resource Contract. Click on Manage Resource contract tab

2. The Search for Resource Contract pop-up is displayed.

3. You can search contract based on the following search criteria:

  • Artist

  • Artist ID

  • Clearance Admin Company

  • Title

  • Version Title

  • ISRC

  • Resource Type

4. You can enter search criteria fields and click Search. The system displays a list of resource contracts in the pop-up.

Note: The linked resource contracts appearing on the search for resource contract pop up are displayed in alphabetical order sorted by artist name, resource title and then Clearance Admin Company.

5. System displays the contract flag icon color as "Green" if contract is linked with the resource in GRS.

6. System displays the contract flag icon as "Red" if no contract is linked with the resource in GRS.

7. You can click the arrow icon against the resource to view the contract details.

8. If the contract is available for the above selected resource in GRS, the system displays the list of contract (s) of the resource as shown below:

9. Select on the relevant contract and click on Link to Workgroup button.

Note:  If the selected contracts do not have the same clearance Admin company, system displays the error message.

10. The Link Contract to Workgroup screen is displayed to link the contract to relevant workgroup.

11. By default, system displays the list of reviewer child workgroups whose company matches with the clearance admin company of the selected contract

12. You can refine the search for workgroups based on one or more combination

  • Search by Workgroup Name

  • Search by Role

  • Search by Company Name and/or ISAC Code

  • Search by User

  • Search by Country / Territory Name

13. Set the flag to select one/more workgroup and click on Link to Contract.

Note:  System links only those contracts which are not previously linked.
