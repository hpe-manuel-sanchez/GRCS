Managing Emails

1. Login to GCS and the home page of the system is displayed.

2. Based on the role, system displays the Email options when you scroll the mouse over/clicks email on the navigation tab. Email option is displayed only when a project is selected in inbox left hand side or the project is open in a new browser.

3. Email Options: System displays the following options when Requestor/Reviewer/RCC Admin scroll the mouse over/clicks email on navigation tab:

  • UPC Allocation

  • General

  • Master

  • TV/Radio Break ICLA

  • Regular/Non-Traditional Non TV/ Radio Break

Note: UPC Allocation option is displayed only for Requestor and Rest all other options are displayed for all the other roles.

4. When you click on any of the email options, the clearance system triggers a new email message to your email id with the relevant details in Subject and Email body pre-populated.

Note: For UPC Allocation, System triggers an email to RCC Admin group with a CC to you. For UPC Allocation, System triggers the mail if the project is regular/non-traditional with release type as new and at least one release on the project should not have UPC.

5. The email content displays the project reference number as hyperlink. You can open the project from the hyperlink based on your access rights to the project.
