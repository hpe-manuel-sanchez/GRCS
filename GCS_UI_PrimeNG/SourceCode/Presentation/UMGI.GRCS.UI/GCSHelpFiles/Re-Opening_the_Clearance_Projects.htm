Re-Opening the Clearance Projects

1. Login to GCS and the home page of the system is displayed.

2. Navigate to the Manage Cancelled and Completed project screen.

3. System displays the Search Cancelled/Completed Projects.

4.  Search for the project based on defined search criteria and click on Project Reference Number to open the project in new browser/tab.

5. System displays the project details with Re-Open button at the bottom of the page.

Note: All the fields are set as non-editable.

6. Click on Re-Open Button, system sets the status of the project to be ‘Reopened’.

Note:

1. After clicking on Re-Open, all project fields become editable on the screen.

2. The status of all the requests remains unchanged.

7. You can edit any of the fields and (re)submit the project.
