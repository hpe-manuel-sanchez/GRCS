Amending Child Workgroups

*Only RCC or user with manage workgroup rights can amend the existing workgroup.

Fields marked with an asterisk * are mandatory fields

1. Login to GCS and the home page of the system is displayed.

2. Select Workgroup -> Manage Workgroup. Click the Manage Workgroup tab.

3. The Maintain Workgroup screen with search options is displayed.

4. You can search the child workgroup through the available search by options, based on either one or combination of the search criteria:

  • Search by workgroup name

  • Search by role in drop-down

  • Search by company name and/or ISAC Company code

  • Search by user name

  • Search by Territory/Country name

5. Enter the search criteria and click Search. The workgroups that match the search criteria are displayed as list in an alphabetical order of workgroup name.

6. After clicking on the Work group Name hyperlink , system opens the Maintain child workgroup screen to amend the child workgroup details in the accordions shown below:

Basic Details

7. All the fields are in editable mode except the Role, Territory/Country, R2 contact, status and Parent workgroups.

8. You can amend the name of the workgroup.

9. To amend company(s) of the child workgroup, click the Manage Company button. The Manage Company pop-up is displayed. For Description: Please Refer Manage Company

Note: After navigating to manage company popup, by default system will display previously added company(s).

10. To amend User (s) of the parent workgroup, click the Manage Users button. The Manage Users pop-up is displayed. For Description: Please Refer Manage Users

Note: After navigating to manage user popup, by default system will display previously added user(s).

Workgroup Deviation for Request Type

11. You can amend the request type for child workgroup by clicking on Workgroup deviation for Request type accordion.

12. You can select or deselect any of the above request types as shown in the above screen.

Workgroup Deviation for Artist Contract

13. You can add/remove artist contract linked to the child workgroup. For description: See Manage Artist Contract

Workgroup Deviation for Resource Contract

14. You can add/remove resource contract linked to the child workgroup. For description: See Manage Resource Contract

15. Click Save. The workgroup details are updated.

Note: By clicking on cancel, system cancels the workgroup amendments.
