Managing Safe Territories

* Only RCC Admin can perform actions on this screen.

1. Login to GCS and the home page of the system is displayed.

2. Select Admin ->Manage Safe Territories. Click on Manage Safe Territories tab.

3. Manage Safe Territories screen is displayed.

4. By default, system displays all the territories as N/A.

Note: The territories already saved as safe territories will appear marked under the safe territories

5. Select the safe territory radio button for the territory that needs to be marked as safe.

6. Selected Safe Territories will get included under Safe Territories List in the right hand side section of the screen.

Note: You can expand the territory to view the list of countries in that territory but there is no selection allowed against the country.

7. Save: On clicking Save, system saves the safe territory list.

Note: By clicking Cancel, system will navigate back to the home page of RCC Admin.
