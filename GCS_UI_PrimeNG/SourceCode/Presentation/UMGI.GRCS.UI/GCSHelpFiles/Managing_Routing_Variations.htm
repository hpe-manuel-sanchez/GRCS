Managing Routing Variations

1. Login to GCS and the home page of the system is displayed.

2. Select Admin ->Manage Routing Variations. Click on Manage Routing Variations tab.

3. Manage Routing Variations screen is displayed. The screen displays all the existing rules in a table with the following details:

4. New Rule: By clicking on New Rule, the Create rule pop up is displayed.

5. Create Rule: Enter the name of the Rule as Free Text and click Create.

Note:The Rule name cannot be edited after create.

6. System displays the Manage Routing Variations screen with the newly created rule.

7. Project Type: Select as desired

8. Request Type: Select as desired

9. Sensitive Artist: You can set the flag Yes/No for Sensitive Artist.

10. Active Artist: You can set the flag Yes/No for Active Artist.

11. Multi Artist: You can set the flag Yes/No for Multi Artist.

12. Compilation:  You can set the flag Yes/No for Compilation.

13. Release Territory: By clicking on Add in Release Territory Column, the Manage Release Territory Popup is displayed. Refer Manage Release Territory screen for the description.

14. Owning Territory: By clicking on Add in Owning Territory Column, the Manage Owning Territory Popup is displayed.

Refer Manage Release Territory screen for the description

15. Owning Company: By clicking on Add in Owning Company Column, the Manage Owning Company Popup is displayed.

Refer Add/Remove a company screen for the description

16. Requesting Territory: By clicking on Add in Requesting Territory Column, the Manage Requesting Territory Popup is displayed.

Refer Manage Release Territory screen for the description

17. Requesting Company: By clicking on Add in Requesting Company Column, the Manage Requesting Company Popup is displayed.

Refer Add/Remove a company screen for the description

18. Talent: By clicking on Add in Talent column, the Manage Artist pop up is displayed.

19. Enter the relevant search field and click Search. System displays the search result for entered criteria.

Refer Manage Artist Functionality screen for the description

20. You can select multiple Artists and click Save.

21. Skip International Marketing: You can set the flag Yes/No.

22. Skip National Marketing: You can set the flag Yes/No.

23. Skip Local Label Marketing: You can set the flag Yes/No.

Note:

1. Yes Option is displayed only for parent rule.

2. No Option is displayed only for Variation.

26. View/Manage : Once the details are entered in any of the owning  territory, requesting territory, requested territory, owning company, requesting company, talent, the add button is changed to view/manage button. The user can edit the previously entered details by clicking on view/manage button.

27. If you have not added details for any one or more of the fields, the clearance system should ignore those for routing calculation to skip local label/marketing steps.

28. Comments: You can add comments against the rule as text. You can click on save button to save the rule at any moment in time

29. Add Variation: By clicking on Add Variation button, system copy the details entered in the main rule in the variations row except the comments

Note: Add variation button is enabled after the user has selected one of the skip steps

30. You can edit any of the details and click on Save.

31. Copy Rule: You can copy any of the existing rule by clicking on Save As. The system will copy the rule and its variations into a new rule. You can give a name to the new rule

32. Activate: You can click on activate link to activate a rule in the routing calculations. When a parent rule is activated, all the associated variations also get activated. When a variation is activated, it has no impact on other variations or parent.
