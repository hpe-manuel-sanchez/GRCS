Free Hand Resource Management

* This is applicable for all the types of project (s).

* For Submission of project with free hand resource (s) , please view section Add freehand resource (Master & Regular).

Note: System send an auto generated email to the R2 administrator when the free hand resource is approved by legal reviewer.

1. On confirmation, Navigate to the requestor inbox.

2. Open the project having free hand resource and navigate to the resource tab .

3. System enables the hyperlink against the resource title for approved free hand resource.

4. Click on Resource title and system opens the pop up “Search for Resource”.Select the resource from R2 which should replace the free hand resource.

For details on search for resource, Please refer Resources Tab.

5. Click Save. System disables the hyperlink in the resource title name.

Note: Linking of free hand resource on the project can be done only when it is approved by reviewer.
