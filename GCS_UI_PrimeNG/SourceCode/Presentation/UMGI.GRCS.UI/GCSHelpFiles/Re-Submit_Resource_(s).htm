Re-Submit Resource (s)

* Re-Submission happens for Submitted/Resubmitted/Re-Opened Project(s)

1. Login to GCS and the home page of the system is displayed.

2. Navigate to the Requestor Inbox and click on the Project Reference Number to view and edit the fields of the projects.

Note:You can open any project with status type Submitted/re-submitted/re-opened.

3. Navigate to the Resource Tab

4. If you wish to edit the Master project resource(s), following are the sections which will trigger Re-Submission

   a. For all Resources

    • Release Countries Include

    • Release Countries Exclude

    • Sensitive Exploitation Flag

    • Excerpt Time – Only when increased

    • Suggested Fee – Only when decreased.

   b. Only Free Hand Resources

    • Artist

    • Title

    • Version Title

    • Recording Type

    • Resource Type

    • Music Type

    • Duration

5. If you wish to edit the Regular/Non-Traditional project resource(s), following are the sections which will trigger Re-Submission

   a. For all Resources

    • Release Countries Include

    • Release Countries Exclude

    • Globally Cleared Flag

   b. Only Free Hand Resources

    • Artist

    • Title

    • Version Title

    • Recording Type

    • Resource Type

    • Music Type

    • Duration

6. On editing fields at the Resource Tab, system displays the below confirmation message:

7. By clicking on Yes, system displays the below confirmation message:

8. By clicking on Yes, then system displays the Resubmit Projects pop up where in it displays the list of rejected and cancelled requests for those resources which have been edited.

9.  You can select one or more or all the requests and click Submit. System resubmits all the selected resources for the selected request types. However, the status of the project does not change.

Note: By clicking on No, system discards all the amendments made to the resource fields.

10. In case, if you edit the comments field on resource tab and click Save. System displays the below confirmation  message:

11. By clicking on Yes, System re-submits only those resource(s) where details were edited.

Note: By clicking on No, system saves the changes to the resource(s) and do not trigger re-submission.
