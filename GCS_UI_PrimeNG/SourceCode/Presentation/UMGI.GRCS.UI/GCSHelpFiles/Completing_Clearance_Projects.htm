Completing Clearance Projects

* User with requestor role and having requestor rights complete projects in Clearance System.

* You can set the project status as complete only for project with status Submitted/Re-submitted/Re-open.

1. Login to GCS and the home page of the system is displayed.

2. Navigate to the Requestor Inbox and click on the Project Reference Number to set the status of the project as complete.

3. System displays the project details with Complete button at the bottom of the page.

4. Click on Complete Button, system sets the status of the project to be ‘Completed’.

Note:1. The project becomes non-editable.  
2. The project is removed from the inbox and is available under cancelled/completed project navigation
3. The status of all the resources pending approval on the project are set to ‘cancelled’.
4. The status of rejected, approved, conditionally approved or cancelled requests   remain unchanged

5. When the project status is changed to complete, the clearance system removes all notifications for pending request(s) on project from the reviewer’s Inbox and sends email to the reviewer informing about project completion.
