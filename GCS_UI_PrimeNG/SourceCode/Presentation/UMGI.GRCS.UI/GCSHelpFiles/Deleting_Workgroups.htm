Deleting Workgroups

*RCC Admin Workgroup can delete a parent or child workgroup.

*Workgroup Users who have manage workgroup rights can delete a child workgroup.

1. Select Workgroup -> Delete Workgroups. Click on Work group tab.

2. The Delete Workgroup screen is displayed.

3. You can search the workgroup through the available search by options, based on either one or combination of the search criteria:

  • Search by workgroup name

  • Search by role in drop-down

  • Search by company name and/or ISAC Code

  • Search by user name

  • Search by Territory/Country name

4. Enter the search criteria and click Search. The active workgroups that match the search criteria are displayed as list in an alphabetical order.

5. Select a workgroup you wish to delete by selecting the option button corresponding to that workgroup.

6. If you wish to view the details of workgroup before deleting it, click View. The View Workgroup pop-up is displayed with workgroup details.

7. You can view the details and click Delete to delete the workgroup or click Cancel to cancel the action.

8. If you wish to delete the workgroup without viewing its details, click Delete in the Delete Workgroup screen.

9. If the selected workgroup is Parent workgroup and if the workgroup was never used in the system for the routing of requests, then

  1. The system deletes the parent and associated child workgroups and removes from the system.

Note: If the selected workgroup is Parent workgroup and if the workgroup was used in the system for the routing of requests, then an error message stating that the workgroup cannot be deleted is displayed.
