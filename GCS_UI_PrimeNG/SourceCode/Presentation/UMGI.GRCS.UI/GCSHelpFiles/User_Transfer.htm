User Transfer

*User in Requestor workgroup/RCC Admin Workgroup can transfer the ownership of the projects to some other user in the same requestor workgroup.

* Only projects with status Submit/Re-submit /Re-open can be transferred for ownership.

1. Login to GCS and the home page of the system is displayed.

2. Select Projects -> Requestor Transfer. Click on Requestor Transfer Tab.

3. The Search Projects for User Transfer screen is displayed.

4. You can enter one or more relevant search criteria.

5. Click Search and system displays the search result as shown below:

Note: By Clicking on Reset, system reset the entered field.
